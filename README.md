# Gamifier

Generate games with no javascript.

## Usage

Run gamifier with this command.

`npx gamifier <game>`

Currently only supported game is `tic-tac-toe`.

### Tic-Tac-Toe Options

Change a player's name:

`npx gamifier tic-tac-toe [p1=<name>] [p2=<name>]`
